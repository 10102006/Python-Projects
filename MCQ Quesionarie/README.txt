 * Summary

- Class Pickle: This class will be used for storing the files and retrieving the files.
- Class Questionarie: This is the main class which will be used to make, store, retrieve and display question on the console

How to use:
    1. Make an instant of questionaire class
    2. For a sample question make a variable {example_question} the use the MakeQuestion() function of the questionaire class. => This will return a dict which will act as an question-object.
    3. Use the DisplayQuestion() for displaying the question in the console.
    4. If the question is running properly the use the SaveQuestion() function to save the question instant in the database folder. => This will an pkl file.
    5. If you want to retrieve the question object use the RetriveQuestion() function. => will return an dict(question-object)
